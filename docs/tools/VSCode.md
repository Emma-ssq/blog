# VSCode

一款轻量级文本编辑器，由于有大量**插件**的存在，使得此编辑器功能强大，**使用方便**，我的代码编辑工作主要在VSCode上。

## 设置

1. 中文版：因人而异
2. 自动格式化

## 常用快捷键

1. 搜索：ctrl+shift+p
2. 设置：直接用鼠标在文件->首选项中点击

## 超实用的插件

### 代码自动补全

Tabnine AI Autocomplete, 一款非常神奇的插件，装上就会爱上。

### markdown

1. **markdownlint**: 自动检查markdown语法并修改。

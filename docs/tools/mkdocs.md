## 链接

MkDocs使用toc扩展名为Markdown文档中的每个header生成一个ID。 您可以使用该ID链接到目标文档中的相应部分。 生成的HTML将正确转换链接的路径，并使锚点来完成跳转。

```bash
Please see the [project license](about.md#license) for further details.

```
